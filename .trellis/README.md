# Trellis project records

The canonical project records in this template are:

- `.trellis/specs/` — product and engineering specifications.
- `.trellis/prds/` — prioritized, executable product requirements.
- `.trellis/research/` — repository-grounded research and baseline reports.
- `.trellis/decisions/` — durable architecture decisions.
- `.trellis/reviews/` — independent review output.
- `.trellis/retrospectives/` — post-merge learning.

The older `.trellis/spec/` and `.trellis/tasks/` paths remain for compatibility with Trellis installations that already consume them. New work should use the canonical paths above.
