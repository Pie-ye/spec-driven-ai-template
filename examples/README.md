# Examples

These are adoption guides, not production applications. Copy the relevant profile directory and adapt the native manifests before enabling it.

- `python-service/` — Python + uv service.
- `kotlin-app/` — Kotlin/Android with a repository Gradle Wrapper.
- `qt-desktop/` — Qt/C++ with CMake Presets and a host Qt SDK.
- `polyglot-product/` — module manifest combining Web, Python, and C/Qt.
