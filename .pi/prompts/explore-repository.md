# Explore repository for a PRD

Read root and affected module `AGENTS.md`, the complete PRD, `.template/profiles.toml`, relevant specs, and native build manifests. Do not edit files.

Return:

- existing architecture and reusable patterns;
- files and symbols relevant to each acceptance criterion;
- current format/lint/typecheck/test/build commands;
- files already satisfying the PRD;
- proposed changes and removals with reasons;
- missing tools, platform constraints, and migration risks;
- a repository baseline path for durable findings.
